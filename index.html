Main components of webpages

  • HTML
  • CSS
  • JavaScript

HTML

HTML is the structure of the webpage. It is the skeleton of the webpage. It is the content of the webpage.

CSS

CSS is the style of the webpage. It is the skin of the webpage. It is the design of the webpage.

Go read the style tag of this page in the header to see how it's being styled.

JavaScript

JavaScript is the behavior of the webpage. It is the muscles of the webpage. It is the interactivity of the webpage.

Javascript code:


// Select the button and paragraph elements
const button = document.getElementById('button-counter');
const counterDisplay = document.getElementById('button-counter-example');

let counter = 0;

button.addEventListener('click', () => {
    counter++;
    counterDisplay.textContent = counter;
});
